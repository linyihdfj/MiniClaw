迪利克雷判别法详细例子
========================

例1：傅里叶级数收敛性
考虑级数：∑[n=1→∞] sin(nx)/n

分析：
设 a_n(x) = sin(nx), b_n(x) = 1/n

1. 部分和有界性：
   利用三角恒等式：
   ∑[k=1→n] sin(kx) = [cos(x/2) - cos((n+1/2)x)] / [2sin(x/2)]
   当 x ∈ [δ, 2π-δ] (0<δ<π) 时，分母有下界：
   |2sin(x/2)| ≥ 2sin(δ/2) > 0
   所以存在 M = 1/sin(δ/2)，使得
   |∑[k=1→n] sin(kx)| ≤ M

2. 单调性：
   b_n(x) = 1/n 是单调递减的正数列

3. 一致收敛于零：
   sup_{x∈[δ,2π-δ]} |1/n| = 1/n → 0 (n→∞)

由迪利克雷判别法，级数 ∑[n=1→∞] sin(nx)/n 在 [δ, 2π-δ] 上一致收敛。

例2：交错级数
考虑级数：∑[n=1→∞] (-1)^n / n

分析：
设 a_n = (-1)^n, b_n = 1/n

1. 部分和有界性：
   ∑[k=1→n] (-1)^k = 
   { 0, 当 n 为偶数
   { -1, 当 n 为奇数
   所以 |∑[k=1→n] (-1)^k| ≤ 1

2. 单调性：
   b_n = 1/n 单调递减

3. 收敛于零：
   lim[n→∞] 1/n = 0

由迪利克雷判别法，级数 ∑[n=1→∞] (-1)^n/n 收敛。

例3：更一般的情况
考虑级数：∑[n=1→∞] cos(nx)/n^p, p>0

分析：
设 a_n(x) = cos(nx), b_n(x) = 1/n^p

1. 部分和有界性：
   利用三角恒等式：
   ∑[k=1→n] cos(kx) = [sin((n+1/2)x) - sin(x/2)] / [2sin(x/2)]
   当 x ∈ [δ, 2π-δ] 时，存在常数 M，使得
   |∑[k=1→n] cos(kx)| ≤ M

2. 单调性：
   当 p>0 时，b_n(x) = 1/n^p 单调递减

3. 一致收敛于零：
   sup_{x∈[δ,2π-δ]} |1/n^p| = 1/n^p → 0 (n→∞)

由迪利克雷判别法，级数在 [δ, 2π-δ] 上一致收敛。

例4：迪利克雷判别法证明思路
证明：设 S_n(x) = ∑[k=1→n] a_k(x)
由条件1：|S_n(x)| ≤ M 对所有 x∈I, n∈ℕ
由条件2：{b_n(x)} 单调
由条件3：b_n(x) → 0 一致

对任意 ε>0，存在 N，当 n>N 时，|b_n(x)| < ε/(2M)
对任意 m>n>N，由阿贝尔求和公式：
∑[k=n+1→m] a_k(x)b_k(x) = S_m(x)b_m(x) - S_n(x)b_{n+1}(x) + ∑[k=n+1→m-1] S_k(x)[b_k(x)-b_{k+1}(x)]

由于单调性，b_k(x)-b_{k+1}(x) 同号，所以：
|∑[k=n+1→m] a_k(x)b_k(x)| ≤ M|b_m(x)| + M|b_{n+1}(x)| + M∑[k=n+1→m-1] |b_k(x)-b_{k+1}(x)|
= M|b_m(x)| + M|b_{n+1}(x)| + M|b_{n+1}(x)-b_m(x)|
≤ 2M|b_{n+1}(x)| + 2M|b_m(x)|
< 2M·ε/(2M) + 2M·ε/(2M) = 2ε

由柯西收敛准则，级数一致收敛。

例5：注意事项
级数 ∑[n=1→∞] sin(nx)/√n 在 x=0 附近不一致收敛，因为：
当 x 很小时，sin(nx) ≈ nx，部分和 ∑sin(kx) ≈ x·n(n+1)/2 无界。

总结：
迪利克雷判别法的核心思想是利用：
1. 部分和的有界性控制振荡
2. 单调性保证求和可控制
3. 趋于零保证余项可忽略
