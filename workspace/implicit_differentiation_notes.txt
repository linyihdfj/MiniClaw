隐函数求导法学习笔记
======================

一、基本概念
隐函数是指函数关系隐含在方程 F(x,y)=0 中，而不是显式地写成 y=f(x) 的形式。

二、隐函数求导的基本方法
1. 对等式两边同时对 x 求导
2. 将 y 视为 x 的函数 y=y(x)
3. 使用链式法则处理包含 y 的项
4. 解出 dy/dx

三、基本例子
例1：圆方程
方程：x² + y² = 1
求导：2x + 2y·dy/dx = 0
解得：dy/dx = -x/y

例2：椭圆方程
方程：x²/a² + y²/b² = 1
求导：2x/a² + (2y/b²)·dy/dx = 0
解得：dy/dx = -(b²x)/(a²y)

四、链式法则的应用
当 y 是 x 的函数时：
1. d/dx [y^n] = n·y^(n-1)·dy/dx
2. d/dx [sin(y)] = cos(y)·dy/dx
3. d/dx [e^y] = e^y·dy/dx
4. d/dx [ln(y)] = (1/y)·dy/dx

五、隐函数求导的一般步骤
1. 写出原方程：F(x,y)=0
2. 两边对 x 求导（将 y 视为 x 的函数）
3. 使用链式法则处理 y 的项
4. 整理方程，将所有含 dy/dx 的项移到一边
5. 解出 dy/dx

六、隐函数定理（理论基础）
设 F(x,y) 在点 (x₀,y₀) 附近连续可微，且 F(x₀,y₀)=0，F_y(x₀,y₀)≠0，
则在 (x₀,y₀) 附近存在唯一可微函数 y=f(x)，满足 F(x,f(x))=0，
且导数：dy/dx = -F_x(x,y)/F_y(x,y)

七、高阶导数求法
对一阶导数表达式再次求导：
d²y/dx² = d/dx (dy/dx)

例：对 x² + y² = 1
一阶导：dy/dx = -x/y
二阶导：d²y/dx² = d/dx(-x/y) = -(y - x·dy/dx)/y²
代入 dy/dx：d²y/dx² = -(y - x·(-x/y))/y² = -(y + x²/y)/y² = -(y² + x²)/y³ = -1/y³

八、多元隐函数求导
对于方程 F(x,y,z)=0，求 ∂z/∂x 和 ∂z/∂y：
∂z/∂x = -F_x/F_z
∂z/∂y = -F_y/F_z
（要求 F_z ≠ 0）

九、常见题型和技巧
1. 求切线方程：先求 dy/dx，再用点斜式
2. 求法线方程：斜率为 -1/(dy/dx)
3. 求曲率：需要一阶和二阶导数
4. 参数方程相关：结合参数求导

十、注意事项
1. 求导时不要忘记链式法则
2. 结果中可能同时包含 x 和 y
3. 检查分母不为零的条件
4. 隐函数定理的条件要满足

十一、练习例子
1. 求 x³ + y³ = 3xy 在点 (3/2, 3/2) 的切线斜率
2. 求 sin(xy) = x + y 的 dy/dx
3. 求 e^(x+y) = xy 的二阶导数
4. 求 ln(x²+y²) = arctan(y/x) 的 dy/dx
