隐函数求导详细例子
====================

例1：基本隐函数求导
方程：x² + y² = 25
求：dy/dx

解：
1. 两边对 x 求导：
   d/dx(x²) + d/dx(y²) = d/dx(25)
   
2. 应用链式法则：
   2x + 2y·dy/dx = 0
   
3. 解出 dy/dx：
   2y·dy/dx = -2x
   dy/dx = -x/y

例2：包含三角函数的隐函数
方程：sin(xy) = x + y
求：dy/dx

解：
1. 两边对 x 求导：
   d/dx[sin(xy)] = d/dx(x + y)
   
2. 应用链式法则和乘积法则：
   cos(xy)·d/dx(xy) = 1 + dy/dx
   cos(xy)·(y + x·dy/dx) = 1 + dy/dx
   
3. 展开：
   y·cos(xy) + x·cos(xy)·dy/dx = 1 + dy/dx
   
4. 整理含 dy/dx 的项：
   x·cos(xy)·dy/dx - dy/dx = 1 - y·cos(xy)
   
5. 提取公因式：
   dy/dx·[x·cos(xy) - 1] = 1 - y·cos(xy)
   
6. 解出 dy/dx：
   dy/dx = [1 - y·cos(xy)] / [x·cos(xy) - 1]

例3：包含指数和对数的隐函数
方程：e^(x+y) = xy
求：dy/dx

解：
1. 两边对 x 求导：
   d/dx[e^(x+y)] = d/dx(xy)
   
2. 应用链式法则和乘积法则：
   e^(x+y)·d/dx(x+y) = y + x·dy/dx
   e^(x+y)·(1 + dy/dx) = y + x·dy/dx
   
3. 展开：
   e^(x+y) + e^(x+y)·dy/dx = y + x·dy/dx
   
4. 整理：
   e^(x+y)·dy/dx - x·dy/dx = y - e^(x+y)
   
5. 提取公因式：
   dy/dx·[e^(x+y) - x] = y - e^(x+y)
   
6. 解出 dy/dx：
   dy/dx = [y - e^(x+y)] / [e^(x+y) - x]

例4：求切线方程
方程：x³ + y³ = 6xy （笛卡尔叶形线）
求：在点 (3,3) 处的切线方程

解：
1. 先求 dy/dx：
   两边求导：3x² + 3y²·dy/dx = 6y + 6x·dy/dx
   
2. 整理：
   3y²·dy/dx - 6x·dy/dx = 6y - 3x²
   
3. 提取公因式：
   dy/dx·(3y² - 6x) = 6y - 3x²
   
4. 解出 dy/dx：
   dy/dx = (6y - 3x²) / (3y² - 6x) = (2y - x²) / (y² - 2x)
   
5. 在点 (3,3) 处：
   dy/dx = (2·3 - 3²) / (3² - 2·3) = (6 - 9) / (9 - 6) = -3/3 = -1
   
6. 切线方程（点斜式）：
   y - 3 = -1·(x - 3)
   y = -x + 6

例5：二阶导数
方程：x² - y² = 1 （双曲线）
求：d²y/dx²

解：
1. 求一阶导数：
   2x - 2y·dy/dx = 0
   dy/dx = x/y
   
2. 对一阶导数再求导：
   d²y/dx² = d/dx(x/y)
   
3. 应用商法则：
   d²y/dx² = [y·1 - x·dy/dx] / y²
   
4. 代入 dy/dx = x/y：
   d²y/dx² = [y - x·(x/y)] / y²
            = [y - x²/y] / y²
            = (y² - x²) / y³
   
5. 由原方程 x² - y² = 1 得 y² - x² = -1：
   d²y/dx² = -1 / y³

例6：隐函数定理应用
方程：F(x,y) = x³ + y³ - 3xy = 0
验证在点 (1,1) 附近存在隐函数 y=f(x)，并求 f'(1)

解：
1. 检查 F(1,1) = 1 + 1 - 3 = -1 ≠ 0，不满足 F(x₀,y₀)=0
   改选点 (3/2, 3/2)：F(3/2,3/2) = (27/8)+(27/8)-3(9/4) = 27/4-27/4=0 ✓
   
2. 计算偏导数：
   F_x = 3x² - 3y
   F_y = 3y² - 3x
   
3. 在点 (3/2, 3/2)：
   F_y = 3(9/4) - 3(3/2) = 27/4 - 9/2 = 27/4 - 18/4 = 9/4 ≠ 0
   满足隐函数定理条件
   
4. 求导数：
   dy/dx = -F_x/F_y = -(3x²-3y)/(3y²-3x) = (y-x²)/(y²-x)
   
5. 在点 (3/2, 3/2)：
   dy/dx = [(3/2)-(9/4)] / [(9/4)-(3/2)] = (6/4-9/4)/(9/4-6/4) = (-3/4)/(3/4) = -1

总结技巧：
1. 始终记住 y 是 x 的函数
2. 对 y 的函数求导要用链式法则
3. 整理方程时，把所有 dy/dx 项集中
4. 结果中允许同时包含 x 和 y
5. 求高阶导数时，先化简再代入
